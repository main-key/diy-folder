

提交流程

	将所有文件提交到工作树
	git add .
	
	将所有文件提交到暂存树，为这次提交添加说明
	git commit -m "添加说明"
	
	提交到仓库
	git push
	
其他命令
	拉取代码
	git pull
	
	查看当前文件夹内文件状态
	git status
	
	查看提交历史
	git log
	
	恢复版本
	git reset –-hard 29d798ab175334848eb6bf5731dac3126d88b207
	
	查看所有分支
	git branch -a
	
	创建分支并切换到该分支
	git checkout -b 分支名
	
	合并分支
	git merge 分支名
	
	提交到主分支
	git push origin master || git push
