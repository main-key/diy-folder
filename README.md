# diy-folder
  windows系统创建DIY文件夹

# 功能
  支持自定义：文件夹别名、文件夹图标、文件夹备注

# 使用方法
  1、将{20230425：V1.2.0}下载到本地指定位置\n
  2、将{MD.exe}文件所在路径配置到环境变量path中[系统变量或用户变量]都可以\n
  3、配置好以后就可以在任意一个文件夹上方的地址栏中输入md或MD回车打开\n
  4、然后根据提示操作即可\n
  注意：设置文件夹图标后图标位置不能生成更改\n
